FastHenry2Object.Stop

void FastHenry2Object.Stop()

Return Value

None

Parameters

None

Remarks

Stops the currently running simulation.

Notes:

It could take some time before FastHenry2 actually stops, depending on the operation currently under execution. You can poll FastHenry2 status checking the IsRunning property or you can set a callback upon FastHenry2 end of operation using the SetEndCallback method.
